Utvecklingsverktyg och plattform
• Visual Studio
• Lämplig SQL-databas, ex vis Microsoft SQL Server
• Lämplig webbläsare
Uppgift
Skapa en kontaktdatabas som innehåller kontaktpersoner på olika företag, kontaktpersoner och företag skall lagras i två separata tabeller i databasen. Varje kontaktperson måste vara kopplad till ett specifikt företag.
En kontaktperson skall ha följande attribut:
• Förnamn
• Efternamn
• E-post
• Telefonnummer
Ett företag skall ha följande attribut:
• Företagsnamn
Skapa sedan en eller flera sidor för att kunna göra följande:
• Det skall vara möjligt att lägga till en ny kontakt.
• Det skall vara möjligt att redigera en befintlig kontakt.
• Visa en lista över samtliga kontakter i databasen, de synliga attributen skall vara: Förnamn, Efternamn, E-post, Telefonnummer, Företagsnamn.
• Om förnamnet för en viss kontakt är lika med ”Nisse” så ska samtliga fält på kontaktens rad i listan visas med rödfärgad text, annars med svart text
• Företag behöver ej hanteras via webbgränssnittet, räcker med att lägga in ett par st. manuellt i databasen.
Övrig information:
• Basera lösningen på MVC
• Fokusera inte på att få med all funktionalitet som beskrivs, lägg istället vikt på att lösa uppgiften på bästa sätt.
• Se detta som en liten del i en större webbplats, lägg därav fokus på objektorientering samt ett enkelt databaslager.
• Använd endast engelska i databasen och koden.
• OBS! Det är ej tillåtet att använda någon form av O/R-mapper i form av Entity Framework eller Nhibernate m.m."
